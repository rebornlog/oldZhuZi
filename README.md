## 📝 文章规范

### 命名规则
```
YYYY-MM-DD-文章主题.md
```
示例：`2026-03-03-小红书变现方向分析.md`

### 元数据格式（文件开头）
```markdown
---
title: 文章标题
date: 2026-03-03
author: 马斯克
tags: [标签 1, 标签 2]
category: articles/notes/projects
---
```

### 文章结构
1. **背景/问题** - 为什么写这篇
2. **分析/过程** - 核心内容
3. **结论/建议** - 可执行的输出
4. **印记** - 马斯克的复盘标记
